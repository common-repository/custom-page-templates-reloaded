Enables you to define a custom template for individual pages.

== Description ==

Extend WP's Page Template functionality by enabling you to build a custom template for your pages. The template is stored along with the post. Possible use cases are for building a landing page, build and process a custom form or anything that is beyound what your current theme can do.

== Installation ==

1. Upload the whole plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. While editing a page, select Custom option from Template dropdown
5. Enjoy!
